Open Source · Unity + Python

Turn a webcam into real-time physics colliders for Unity

Depth Anything V2 estimates metric depth from any camera feed. A Python server streams it to Unity over TCP, which generates voxel-based collision meshes — floors, walls, furniture — at interactive frame rates.

Pipeline

How it works

Webcam → Depth Anything V2 → TCP Stream → Unity Voxel Colliders
01 Capture

Depth estimation

Python captures webcam frames and runs Depth Anything V2's metric model, producing per-pixel distance maps in real time.

Python · PyTorch · OpenCV
02 Stream

Binary TCP protocol

Depth maps, compressed color frames, and camera intrinsics are packed into a binary stream over a local TCP socket.

TCP · Binary protocol · JPEG
03 Collide

Voxel physics generation

Unity back-projects depth to 3D, voxelizes the scene, runs greedy box merging, and spawns optimized colliders. Auto-leveling extends the floor.

Unity · C# · Greedy merge
Features

What's inside

3D greedy box merging

Voxel collision with greedy merge across all three axes. Produces the minimal set of axis-aligned box colliders for maximum physics performance.

Auto floor leveling

Least-squares plane fitting detects the floor automatically and corrects camera tilt so physics stays level with a handheld camera.

Real-time visual mesh

Textured 3D mesh reconstructed each frame with UV mapping and depth-discontinuity filtering for clean triangulation.

Active ragdoll system

Muscle-driven ragdoll with joint rotation followers, hip tracking, and torso stabilization for physics character interaction.

Configurable pipeline

Model size, indoor/outdoor scenes, voxel resolution, depth range, and preview modes — all via command line flags.

Auto-reconnect TCP

Streams intrinsics, float32 depth, and JPEG color in a single packet. Reconnects automatically on disconnect.

Get started

Quick start

Python side

  • Python 3.8+
  • PyTorch (CUDA recommended)
  • OpenCV
  • Depth Anything V2 weights

Unity side

  • Unity 2021.3+
  • Standard 3D project
  • No extra packages needed

1 Clone the repo

Terminal
git clone https://github.com/SamuelL1234/DepthToArColliders.git
cd DepthToArColliders

2 Download model weights

Place Depth Anything V2 metric checkpoints in checkpoints/. See the Depth Anything V2 repo for download links.

3 Start the depth server

Terminal
python depth_stream.py --source 0 --model-size base --preview

# Options:
#   --source 0           Webcam index or video path
#   --model-size         small | base | large
#   --scene              indoor | outdoor
#   --port 9999          TCP port for Unity
#   --depth-width 320    Depth output resolution
#   --preview            Show depth visualization

4 Open the Unity scene & hit Play

Open Scenes/SampleScene.unity and press Play. Unity auto-connects to the Python depth server on 127.0.0.1:9999.
